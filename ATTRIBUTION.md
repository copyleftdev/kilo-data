# Dataset attribution

KiloCheck compiles data published by IANA, AFRINIC, APNIC, ARIN, LACNIC,
RIPE NCC, the Spamhaus Project, abuse.ch, the Tor Project, Amazon Web Services,
Google, and Cloudflare.

Spamhaus DROP data is © The Spamhaus Project SLU and is redistributed with its
source metadata, date, copyright, terms link, and attribution intact in the
KiloCheck snapshot manifest. See https://www.spamhaus.org/drop/terms/.

Feodo Tracker data is provided by abuse.ch under CC0.

Provider, registry, routing, and anonymity data is contextual. Its presence is
not a claim that an address is malicious.

DShield and the MISP warning-list collection are not included in public release
artifacts pending final licensing and upstream-lineage review.
